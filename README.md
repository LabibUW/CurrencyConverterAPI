# Currency Converter API
Use this RESTful API in order to list ALL of the possible currencies in JSON. (Updated in July 2021)

## Technologies Used
<ul>
  <li>Node.Js</li>
  <li>MongoDB</li>
</ul>  
Node.Js Imports <br />
<ul>
  <li>Mongoose</li>
  <li>Nodemon</li>
  <li>Body-Parser</li>
  <li>Express</li>
  <li>Morgan</li>
</ul>  

## API Structure
GET /currencylist <br />
  "_id": "", <br />
  "abbreviation": "", <br />
  "name": "",
  

## Host this API locally!
Clone the repo
```
npm install
npm start
```
